# Openstaller Roadmap

## Near Term

- archive emitters for `.zip` and `.tar.gz`
- per-file uninstall journal
- generated script dry-run mode
- manifest schema versioning tests
- macOS `.app` launcher metadata
- registry and desktop-entry integration tests

## Systems Track

- additional Assembly kernels for ARM64
- optional signed manifest verification
- real certificate-backed release signing policy
- deterministic archive ordering
- C API for embedding Openstaller into other build tools

## UX Track

- richer Win32 validation states
- generated package preview
- license preview pane
- install path policy presets
- import/export of project profiles
